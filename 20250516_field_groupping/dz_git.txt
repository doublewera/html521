1. Скачать из чата и установить git тем, кто не со своим ноутбуком на уроке
2. открыть НОВУЮ комндную строку cmd (пуск, поиск, cmd) и написать там 
git --version
Оно ответит
git version 2.42.0.windows.2

3. Там же в командной строке написать
ssh-keygen
и на все вопросы ответить ENTER

4. Появилась папка
C:\Users\Student\.ssh

а в ней файл 
id_ed25519.pub

ОТКРЫТЬ ЕГО ТЕКСТОВЫМ РЕДАКТОРОМ! НЕ ДВОЙНЫМ КЛИКОМ!
Сначала открываем текстовый редактор, потом ищем файл
Если блокнот, то переключаем на "все файлы"

5. Скопировать строку  (СВОЮ!!!) - считайте это пара "пароль-отзыв"
ssh-ed25519 AAAAC3NzaCБУБУБУБУБУБУБУБУБУБУБu1UBRT3V7U2Zl9/MwOZБОБОБОБОБОБОp9H 

6. На гитхабе входим в аккаунт

7. Справа, где кружочек с аватаркой или картонкой с квадратиками выбираем SETTINGS (шестереночка)

8. СЛЕВА выбираем пункт SSH GPG....

9. НОВЫЙ (NEW SSH KEY)

10. В заголовке пишем "мой домашний комп"
В поле для ключа вставляем ключ, нажимаем соранить, добавить

11. Клонируем репозиторий
Для этого открываем "мои репозитории" и в кнопке "код" выбираем SSH

12. В командной строке набираем

git clone git@github.com:ВАШ_ЮЗЕРНЕЙМ_НА_ГИТХАБЕ/html521.git

13. git config user.name --local "ВАШЕ ИМЯ"
14. git config user.email --local "ВАШ ЕМАЙЛ"
