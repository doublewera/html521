Ветвление if-else и цикл while

if - если

else - иначе

while - пока

Ветвление позволяет компьютеру принимать решения. Решения принимаются на основе логических выражений. Как написать логическое выражение "Зовут ли этого человека Вася?"

    
    const imya = 'Вася'
    const name1 = 'Коля'
    const imyaVasya = imya == 'Вася'  // Правда ли, что imya - Вася?
    console.log('Правда ли, что imya - Вася?', imyaVasya)
    const name1Vasya = 'Вася' == name1  // Правда ли, что name1 - Вася?
    console.log('Правда ли, что name1 - Вася?', name1Vasya)

Как и у number, range можно попросить сразу отдать число: valueAsNumber. Логическое выражение после if берется в скобки! Действия, выполняющиеся в случае ИСТИНЫ берутся в ФИГУРНЫЕ скобки ПОСЛЕ круглых отступ (в JS!!) не является синтаксическим элементом, но является важным визуальным элементом 

    
    function check() {
    console.log(age.value)
    console.log(age.valueAsNumber)
    if (age.valueAsNumber >= 7) {
        if (age.valueAsNumber >= 18) {
            if (age.valueAsNumber >= 25) {
                if (age.valueAsNumber >= 55) {
                    // дальше пенсионер
                } else {        
                    // до 55 работник
                }
            } else {
                // до 25 студ
            }
        } else {
            // до 18 шк
            console.log('Школьник')
        }
    } else {  // Меньше семи!
        // Ветка ложного случая 
        console.log('Дошкольник')
    }
    // Добавить другие возрастные категории
}

Ветка ложного случая НЕОБЯЗАТЕЛЬНАЯ, её можно не писать, тогда в ложном случае не будет сделано ничего Фигурные скобки называются БЛОКОМ. Они НЕОБЯЗАТЕЛЬНЫ, если потом идет ОДНА команда ЖЕЛАТЕЛЬНО их писать ВСЕГДА!

Цикл - действие, которое повторяется несколько раз, либо по условию. Условие может изначально быть ложным, тогда цикл не выполнится ни разу.

Рассмотрим задачу: Написать страницу, поле ввода конфет "вазочку" и вывести в консоль по одной конфете "съел", пока они не кончатся, т.е. пока их количество не станет равным нулю, на практике в консоли появится мгновенно вывод "обратного отсчета" до нуля 

    <input id="candies" type="number">
    <input type="button" onclick='eat()' value="Сожрать все">
    
    function eat() {
    console.log('Всего конфет было: ', candies.valueAsNumber)
    while (candies.valueAsNumber > 0) {  
        // Продолжать, ПОКА условие истинно
        console.log('Взял одну, осталось: ', --candies.valueAsNumber)
    }
    // Проверяет условие, выполняет тело
}

Задачи

  1. Ветвление: Напишите программу, в которой есть количество денег и цена товара. Например, букет цветов стоимостью 4500 и в кошельке 6000. Программа должна напечатать в консоль "достаточно", если денег хватит и "мало", если нет. НИКАКОЙ СТРАНИЦЫ И КНОПКИ! Только прикрепить скрипт. ОДНА переменная на цену, одна на кошелек, одно ветвление.
  2. Цикл: Продавец и покупатель торгуются на рынке. Продавец изначально называет цену: 10000 за букет. Покупатель 4000. На каждом витке цикла продавец уменьшает сумму на 500, а покупатель повышает на 600. За какую сумму будет куплен букет?
  3. Ветвление: в программе заданы два числа. Если первое можно поделить на второе, напишите их частное, если нельзя (второе число равно нулю), напишите "на ноль делить нельзя". Например, если первое число равно 6, а второе 2, то написать 3, а если второе число 0, то написать "на ноль делить нельзя".
  4. Цикл: напечатайте в консоль: 3*2=6, 3*3=9... 3*9=27 в столбик.
  5. Цикл: на странице есть поле ввода и кнопка. Напечатайте в консоль: x*2=..., x*3=... x*9... в столбик. x берется из поля ввода.
  6. Цикл: напечатайте в консоль всю таблицу умножения, начиная с 1*1 до 10*10
  7. Цикл: напечатайте в консоль всю таблицу умножения, начиная с 1*1 до x*x, где x берется со страницы
  8. Цикл: напечатайте НА СТРАНИЦУ всю таблицу умножения В ВИДЕ ТАБЛИЦЫ, начиная с 1*1 до 10*10
  9. Сделать на странице поле ввода и кнопку. Если в поле ввода четное число, написать в консоли "чётное", если нет - нечётное. Вам понадобится оператор взятия остатка от деления. 5%2 будет 1 (нечётное), 10%2 будет 0 (чётное)
  10. Сделать на странице три поля ввода. Если они могут быть сторонами треугольника, написать в консоль "могут", иначе "не могут". В треугольнике сумма любых двух сторон всегда БОЛЬШЕ третьей стороны.
  11. Сделать на странице три поля ввода (a, b, c). Если они могут быть в этом порядке коэффициентами квадратного уравнения с двумя корнями в вещественном случае, написать "2", если корень 1, написать "1", иначе написать "нет корней". Квадратное уравнение имеет 2 корня, если дискриминант D=(b*b-4*a*c) больше нуля и один корень, если равен нулю. Если дискриминант отрицательный, квадратное уравнение не имеет вещественных корней.
  12. Сделать на странице поле ввода количества элементов. Создать динамически список, в котором столько элементов, сколько ввели в поле ввода. Например, если ввели 3, то список из трёх элементов.
