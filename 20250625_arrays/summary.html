Строки

Строки состоят из символов!

Чтобы обратиться к конкретному символу, надо поставить после строки квадратные скобки и внутри них - номер символа в строке

    
    const imya = 'Вася'
    console.log(imya[1])

Символы в строке нумеруются с нуля! Строка знает свою длину.

Пример 1: вывести все буквы имени.

    
    const imya = 'Вася'
    let bukva_nomer = 0  // Чтобы начать с начала, нужно присвоить букве номер ноль
    while (bukva_nomer < imya.length) {  // пока номер буквы меньше длины строки
        console.log(imya[bukva_nomer])   // печатаем номер буквы
        bukva_nomer++                    // не забываем увеличивать номер буквы
    }

У каждого символа есть свой номер в таблице кодировки. Кодировки бывают разные, но латинские буквы часто имеют те же номера в разных кодировках. 

    
    const myletter = 'B';
    console.log(myletter.charCodeAt(0))
    console.log(String.fromCharCode(97))
