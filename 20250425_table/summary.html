CSS

Для отображения элементов можно использовать таблицу стилей CSS. Таблица стилей состоит из селекторов и правил. Селекторы:

  • tag-name (имя тэга)
  • .className (имя класса)
  • #id (идентификатор, уникальное название)

table {
        border: 10px;
        border-color: black;
        width: 400px;
        height: 400px;
        border-style:groove;
    }
    td {
        border: 1px;
        border-style: solid;
        border-color: black;
    }
    .unknown {
        background-color: rgb(213, 164, 157);
    }
    .known {
        background-color: aquamarine;
    }

В примере выше: table, td - имена тэгов, unknown, known - имена классов

В фигурных скобках - правила. Пары "свойство-значение", разделённые двоеточием. Между правилами пишется точка с запятой.

Примеры правил: background-color - цвет фона. border - рамка. width, height - ширина и высота

Как добавить таблицу стилей?

  1. Создать файл appearance.css (или другое название)
  2. Написать в нём одно правило: 
            body {
            background-color: blue;
        }
  3. Добавить в свою html-страницу строчку в голову:
    4. <link rel="stylesheet" href="appearance.css">

С чего вообще начать???

  1. Прочесть задание
  2. Создать папку с сегодняшней датой и темой 20250425_styles
  3. В ней создать два файла: page.html, appearance.css
  4. Если такие файлы уже есть, придумать другие названия
  5. В задании наверняка указано, ЧТО рисовать и КАК оно должно выглядеть
  6. ЧТО - В HTML! выбираем tag:
    • таблица - table (внутри tr, td)
    • список с циферками? - ol
    • список с точечками? - ul
    • заголовок? - h1, h2...
    • абзац? - p
  7. КАК рисовать? - в CSS!
    • цвет - color, background-color, border-color
    • ширина - width
    • высота - height
    • край, рамка - border
    • тип края - border-style
    • тип края - border-style
